# 行为准则

我们希望这个仓库保持一个尊重、克制、可协作的开源环境。

## 我们鼓励的行为

- 尊重他人时间和表达
- 聚焦问题本身，不做人身攻击
- 以可复现、可验证的方式讨论技术问题
- 接受反馈，也愿意修正自己的错误
- 让讨论对项目推进有帮助

## 不接受的行为

- 骚扰、威胁、歧视、羞辱
- 持续性的恶意挑衅或引战
- 泄露他人隐私信息
- 在 issue、PR、讨论区进行敌意交流
- 任何明显破坏社区协作秩序的行为

## 适用范围

本准则适用于：

- GitHub Issues
- Pull Requests
- Discussions
- 项目文档
- 与本项目直接相关的公开交流场景

## 举报方式

如遇到违反本准则的行为，请联系：

- `heyug3@gmail.com`

如果可以，请附上：

- 发生了什么
- 发生位置
- 链接、截图或其他证据

## 维护者处理方式

维护者会根据情况决定处理措施，包括但不限于：

- 提醒或要求修正
- 删除评论、Issue、PR 或相关内容
- 临时限制参与
- 永久封禁

处理目标是保护协作环境，而不是扩大冲突。
