# 安全说明

## 支持范围

当前只为仓库最新版本处理安全问题。历史旧版本一般不单独回补。

## 漏洞反馈方式

**不要通过公开 GitHub Issue 提交安全漏洞。**

如果你发现了安全问题，请私下发送邮件到：

- `heyug3@gmail.com`

建议在邮件里带上：

- 漏洞描述
- 复现步骤
- 潜在影响
- 如果你已经有判断，也可以附上修复建议

## 覆盖范围

本说明主要覆盖当前仓库中的源码与脚本，包括：

- `skills/ppt-master/scripts/` 下的脚本
- SVG 后处理与 PPTX 导出链路
- 项目管理与文档转换脚本

以下内容不在本仓库安全响应范围内：

- 第三方 AI IDE 或第三方模型平台
- 用户自己提供的源文件
- 已生成的 PPT / DOC / XLS 产物本身

## 处理原则

- 默认按负责任披露原则处理
- 收到反馈后会尽快确认问题
- 修复可公开时，再统一说明或发布更新
